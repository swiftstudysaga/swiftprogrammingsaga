C Program: Calculate Area of a Circle

Explanation

This program calculates the area of a circle using the formula:

area = π × radius × radius

The user is prompted to enter the radius, and the program computes the area using a predefined constant for π (3.14).

  • The scanf function is used to read the radius from the user.
  • The printf function displays the calculated area.
  • The float data type is used for the area to allow decimal values.
  • The result is formatted to two decimal places using %.2f.

Code Example


#include <stdio.h>
#define PI 3.14

void main() {
    int radius;
    float area;

    printf("Enter the radius of the circle: ");
    scanf("%d", &radius);

    area = PI * radius * radius;
    printf("The calculated area of the circle is: %.2f\n", area);
}

How to Compile and Run

Follow these steps to compile and run the program:

  1. Open your terminal.
  2. Navigate to the directory containing your C file.
  3. Run the following commands:

gcc -o circle_area circle_area.c
./circle_area

Note: Replace circle_area.c with the actual filename of your C source code.
