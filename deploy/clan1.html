C Program to calculate area of circle

Explaination

This program calculates the area of a circle using the formula area = π * radius * radius. The user is prompted to enter the radius, and the program computes the area using the defined constant for π (3.14).

The program uses the scanf function to read the radius from the user and the printf function to display the calculated area.

Note: The program uses the float data type for the area to allow for decimal values, and the result is formatted to two decimal places using %.2f.

            
      1 | #include <stdio.h>
      2 | #define PI 3.14
      3 |
      4 | void main() {
      5 |     int radius;
      6 |     float area;
      7 |
      8 |     printf("Enter the radius of the circle: ");
      9 |     scanf("%d", &radius);
     10 |
     11 |     area = PI * radius * radius;
     12 |     printf("The calculated area of the circle is: %.2f\n", area);
     13 | }

gcc -o circle_area circle_area.c
./circle_area
